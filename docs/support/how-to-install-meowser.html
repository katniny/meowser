How to Install Meowser

How to Install Meowser on All Platforms





Windows

How to install Meowser on Windows

  • Go to the GitHub Release Page
  • Click on "Meowser-Setup-vNUM.NUM.NUM.exe
  • Go to your Downloads folder (or wherever you installed it) and run the installer
  • After installation, Meowser should automatically launch!

    • Enjoy your brand new browser!



    macOS

    How to install Meowser on macOS
