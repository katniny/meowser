Meowser Links

All the internal links that Meowser offers.


meow://settings
Allows to modify your Meowser experience.

meow://credits
See everyone who helped with Meowser, as well as the open source projects Meowser relies on.

meow://history
See your browser history, you can also delete it.

meow://extensions
See your installed extensions, as well as add or delete them.

meow://help
The Meowser help page.

meow://updates
See update notes, see the browser's current version and check for updates.

meow://feedback
Help improve Meowser by providing us feedback!

meow://about
All the Meowser internal meow:// links.

meow://developer-tools

Opens the dev tools for the current tab.
*Please type this manually into the URL bar. We will not automatically open this.

meow://more
See more products that Katniny Studios has released.
