Meowser Context Menus

The context menus for Meowser, why are they a work-in-progress?


Why?

Context menus meet certain requirements, and with how Meowser is built, it would require us to make them from scratch as it is.

As a lot of Meowser features are not used from other browsers--only limited features such as the rendering engine--are used. Most of Meowser is writen from scratch using HTML, CSS and JavaScript.

We use Chromium (with Electron), as you may know, and the Chromium context menus are not supported natively, as tabs, the URL bar, bookmarks, etc. are not either, and are written from scratch as well.



So what is supported?

The current HTML elements that are supported for Meowser's context menus are:

  • anchor (/a/)
  • image (/img/)
  • video (/video/)


What elements need support still?

These elements need coded in or investigated for special context menus. We accept all and any contributions!
