Thank you for checking out Meowser!


Meowser is in very early development, and does not have everything you would expect from a browser yet.

We're working on it! If you're interested in contributing, we'd really appreciate it!


This is an in development build. The browser itself, Meowser Shield (the built-in content blocker), and everything else is still under development.

Meowser will be constantly improved with more features, settings, and more.


Even though Meowser is still in development, you can browse the internet as you would be able to normally! If you find a web incompatibility, please let us know on our GitHub Issues page.

(Please note I'm a solo 16 year old developer trying to manage multiple ambitious projects but I'll try to update Meowser constantly!)
